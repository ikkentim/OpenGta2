## Data types
- *.sty*: Style graphics file
- *.GMP*: Map file
- *.SEQ*: Map composition file
- *.SCR*: Script file
- *.MMP*: Multiplayer map file
- *.GXT*: Game text file
- *.GCI*: Vehicle parameters file
